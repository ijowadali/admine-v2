# Travel CRM

## Environments

```
1. Production

==> Branch Name: main

2. Development

==> Branch Name: develop

```

## Install

```sh

git clone https://github.com/jawad2020-web/admine-v2.git
cd admine-v2
yarn install
```

## Start development

```sh
yarn dev
```

## Build for production

```sh
yarn build
```
